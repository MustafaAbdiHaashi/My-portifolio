# Mustafa's Portfolio

A premium modern 3D animated personal portfolio website built with React, Three.js, and Framer Motion.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Three.js** (@react-three/fiber, @react-three/drei) (3D Elements)
- **Lucide React** (Icons)

## Features

- Dark theme with animated gradient background
- Glassmorphism navbar
- 3D interactive floating hero object
- Animated skill cards
- Project showcase with tilt effect
- 3D Earth model in contact section
- Smooth scroll navigation
- Responsive design

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Customization

- Update `src/constants/index.js` to change the content (projects, skills, about).
- Modify `src/assets` to include your own images.
- Adjust `src/styles.js` for global style tweaks.
